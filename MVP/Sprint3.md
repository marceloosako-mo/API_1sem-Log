#   MVP - API Grupo G4 Logística
##   Objetivo do MVP
- Criar uma dashboard de BI para visualizar o desempenho de Estados e municípios paulistas no comércio exterior a partir de dados do MDIC.

---

##   Descrição da Solução
- Evolução histórica da balança comercial
- Identificar países emergentes
- Impacto de políticas econômicas
- Vias de transporte e logística
- Impacto logístico nos custos


---

##   User Stories (Backlog do MVP)
| ID  | User Story                                                                 | Prioridade | |
|-----|-----------------------------------------------------------------------------|------------|------------|
| US03| Como analista/gestor, quero ver a evolução histórica da balança comercial dos municípios de SP, para analisar tendências ao longo do tempo.| Alta    | |
| US08 | Como analista/gestor, quero identificar países com crescimento nas importações/exportações com os municípios paulistas, para identificar mercados emergentes.| Média| |
| US09 | Como analista/gestor, quero analisar o impacto de políticas tarifárias ou acordos comerciais, para entender os efeitos no comércio local. | Baixa     | |
| US10| Como analista/gestor, quero visualizar as principais vias de transporte utilizadas para exportação/importação, para avaliar a eficiência logística.|Baixa||
| US11 | Como analista/gestor, quero entender como a logística impacta os custos e a competitividade dos municípios no mercado internacional.     | Baixa     | |
